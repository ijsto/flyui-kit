# Quickstart

## Installation

To install run `yarn add flyui-kit` or `npm i flyui-kit` on your project.

## Using components

Import the components and use them as you normally would.

```js
import { Avatar, Box, Button, Input, Tag, Text } from 'flyui-kit';

const FlyDemo = () => {
  return (
    <Box borderRadius={3} border={2} p={4} maxWidth={700} mx="auto">
      <Avatar
        src="https://source.unsplash.com/92x92/?portrait"
        shape="round"
        outline
      />

      <Button
        hoverEffects="shadow-large lift"
        variant="outline"
        startIcon="edit"
      >
        Fly Button
      </Button>
    </Box>
  );
}
```
