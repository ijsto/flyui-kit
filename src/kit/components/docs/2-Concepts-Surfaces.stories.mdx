# Surfaces

Surfaces are containers that are placed on top of the background or other canvases.

## Surface Components
- Card
- Modal

## Customizing

The above example will change the surface property via CSS variables.

If you prefer customizing via the `theme`, add `theme.surface.radius` (remember - CSS variable (in this case `--radius-surface`) will take precedence).

You can further control the radius (or many others properties) by directly passing a prop to the component instance where you use it.

The accepted values are either:
- the name of the radius from the predefined theme values (for example: "max")
- an Integer representing the theme space values index

or 

- a simple CSS value:

```jsx
import { Modal } from 'flyui-kit';

const YourComponent = (props) => (
  <Modal borderRadius="max" {...props}>
    {props.children}
  </Modal>
);
```
